Tổng quan về định dạng JSON

JSON là một định dạng dữ liệu ban đầu nhằm hiển thị nội dung của các đối tượng JavaScript.JSON là từ viết tắt của JavaScript Object Notation.Hiện nay định dạng này được sử dụng rộng rãi như một định dạng lưu trữ và trao đổi dữ liệu qua Internet.

JSON cho phép bạn lưu trữ dữ liệu có cấu trúc ở dạng văn bản.

Tất cả các biến trong JSON, ngoại trừ giá trị mảng, cần có tên.

Tên cần được đặt trong chuỗi.

Giá trị được phân tách với tên bằng dấu :

Các biến được phân tách với nhau bằng dấu phẩy.

Ví dụ:

"Name":"Đại đế Fred"
"Age":26

Mảng trong JSON được chỉ định bằng dấu ngoặc vuông:

"My lovely fruits":["banana", "apple", "orange"]

Các đối tượng trong JSON được chỉ định bằng dấu ngoặc nhọn:

{ 
  "name":"Harry",
  "family":"Петров",
  "age":21,
  "girlfriends":
  [
     { "name":"Maria ozawoa",
       "tel":"+79442265612"
     },
     { "name":"Yua",
       "tel":"+79529021928",
       "age":20
     } 
  ]
}
