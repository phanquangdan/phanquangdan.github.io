Đề bài:

Nhập một số nguyên từ bàn phím. Đảo các bit của byte đó của số gốc, số được nhập từ bàn phím và hiển thị số kết quả trên màn hình. Đánh số byte bắt đầu từ 0 (0 - byte thấp, v.v.).

Định dạng đầu vào:số nguyên dương 4 byte (thập lục phân) và số byte (thập phân).

Định dạng đầu ra:số nhận được (với các bit của byte được chỉ định đảo ngược) ở dạng thập lục phân.

Ví dụ (dòng đầu tiên là đầu vào của chuỗi nguồn, dòng thứ hai là đầu ra của kết quả).

Code :

#include <iostream>
using namespace std;
typedef unsigned char uchar;
int main() {
	unsigned int x;
	int n;
	uchar *p = (uchar*)&x;
	cin >> hex >> x;
	cin >> dec >> n;
	unsigned int mask = 0xff;
	mask = mask << (n * 8);
	cout << hex << (x^mask) << endl;
	return 0;
}
