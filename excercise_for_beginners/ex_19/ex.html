Đề bài:

Nhập một số nguyên từ bàn phím.Nếu byte cao của số này lớn hơn byte thấp, hãy đặt lại 4 bit cao của mỗi byte bằng 0.Nếu không, hãy đặt lại 4 bit thấp của mỗi byte bằng 0.Hiển thị kết quả trên màn hình.

Định dạng đầu vào:Số nguyên dương giá trị 4 byte(hệ thập lục phân)

Định dạng đầu ra:Số thập lục phân.

Ví dụ (dòng đầu tiên là đầu vào của chuỗi nguồn, dòng thứ hai là đầu ra của kết quả).

Code :

#include <iostream>
using namespace std;
typedef unsigned char uchar;
typedef unsigned int uint;
int main() {
	uint x;
	uchar *p = (uchar*)&x;
	cin >> hex >> x;
	uint mask_1 = 0x0f0f0f0f, mask_2 = 0xf0f0f0f0;
	if ((int(*(p+3)))>(int(*p)))
		x=(x& mask_1);
	else 
		x=(x& mask_2);
	cout << hex << x << endl;
	return 0;
}
