Đề bài:

Nhập một số nguyên từ bàn phím.Nếu bit thứ 3 của byte cao của số này bằng 1 và bit thứ 4 của byte này được bằng 0, hãy đặt lại byte thấp của số đã nhập bằng 0, nếu không, hãy giữ nguyên số đã nhập không thay đổi. Hiển thị kết quả trên màn hình.

Định dạng đầu vào:Số nguyên dương giá trị 2 byte(hệ thập lục phân)

Định dạng đầu ra:Số thập lục phân.

Ví dụ (dòng đầu tiên là đầu vào của chuỗi nguồn, dòng thứ hai là đầu ra của kết quả).

Code :

#include <iostream>
using namespace std;
typedef unsigned char uchar;
int main() {
	unsigned short x;
	uchar *p = (uchar*)&x;
	cin >> hex >> x;
	if (((x & (1 << 12)) == 0) && (x & (1 << 11)))
		*p = 0;
	cout << hex << x << endl;
	return 0;
}
