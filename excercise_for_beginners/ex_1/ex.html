Đề bài:

Đối với một số nguyên dương được nhập từ bàn phím, hãy hiển thị số đầu và số cuối của nó.

Định dạng đầu vào : số nguyên dương 4 byte.

Định dạng đầu ra : Chữ số đầu tiên và chữ số cuối cùng, được phân tách bằng dấu cách.

Ví dụ (dòng đầu tiên - đầu vào số, thứ hai - đầu ra kết quả):



Code:

#include <iostream>
using namespace std;
int main() {
	int x;
	int l;
	cin >>x;
	l=x%10;
	while (x > 9)
		x = x/10;
	cout << x << " " << l << endl;
	return 0;
}
