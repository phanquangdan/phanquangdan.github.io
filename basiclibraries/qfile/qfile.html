Danh sách các chuỗi và tệp.

Thường thì một lập trình viên phải lưu và tải dữ liệu văn bản từ một tệp. Thư viện Qt để làm việc với tệp có lớp QFile, sẽ được thảo luận chi tiết hơn ở phần sau.Chúng ta sẽ làm quen với lớp bổ trợ QTextStream và xem xét mã tải văn bản từ một tệp.

Để làm việc với tệp và luồng văn bản,sẽ bao gồm các mô-đun tiêu đề bắt buộc:

#include <QFile>
#include <QTextStream>

Mã để tải văn bản từ một tệp vào QStringList sẽ trông giống như sau:

 // khai báo một biến - một danh sách các chuỗi     
    QStringList sl ;  
    // Khai báo một biến tệp và đặt tên tệp        
    QFile file( "c: \\ test.ini" );  
    // Đang cố gắng mở tệp   
    if ( file . open ( QFile :: ReadOnly | QFile :: Text )) {   
        // Tạo một luồng văn bản , chuyển cho nó một con trỏ tệp        
        QTextStream ts (& file ); 
        // Cho đến khi tệp kết thúc    
        while (! ts.atEnd ( ) ) { 
            // đọc dòng tiếp theo và đưa nó vào danh sách các dòng         
            sl << ts . readLine ();  
        }
        // Đóng tệp  
        file . close();
   }

Lưu kết quả vào một tệp có thể trông giống như sau:

 // Khai báo một biến tệp và đặt tên cho tệp đã lưu        
    QFile resFile ( "c: \\ Temp \\ result.txt" ); 
    // Đang cố gắng mở tệp để ghi     
    if ( resFile . open ( QFile :: WriteOnly | QFile :: Text )) {    
        // Tạo một luồng văn bản , chuyển cho nó một con trỏ tệp        
        QTextStream ts (& resFile ); 
        // Đối với mỗi dòng từ danh sách     
        for ( auto line: sl ) {   
            // đưa nó vào dòng văn bản và thêm "end of line"         
            ts << line << '\ n' ;    
        }
        // Đóng tệp  
        file . close ();
    }








Lớp QFile cho phép bạn thực hiện các thao tác cơ bản với tệp - sao chép, di chuyển , xóa tệp:








 /// Sao chép


  QFile origFile("C:\\TEMP\\result.txt");




  if (origFile.exists()) {




    origFile.copy("d:\\Data\\old-result.bak");




  }




  /// Đổi tên 




  QFile file("C:\\TEMP\\result.txt");




  if (file.exists()) {




     file.rename("C:\\Data\\result.txt");




  }




  /// Xóa




  QFile oldFile("C:\\TEMP\\result.txt");




  if (file.exists()) {




     file.remove();




  }


 







Lớp QFileInfo

Lớp QFileInfo cho phép bạn nhận thông tin mở rộng về tệp: liệu có tệp trùng tên hay không, tìm kích thước, ngày tạo và sửa đổi tệp, đường dẫn đầy đủ, v.v. Lớp này không cho phép các thao tác với tệp, nói một cách đại khái, nó có quyền truy cập chỉ đọc vào hệ thống tệp.








  QFileInfo fInfo("c:\\Temp\\test.mp3");
  if (fInfo.exists()) {




    fInfo.size(); /// Kích thước file tính bằng byte




    fInfo.fileName(); /// chỉ tên file - test.mp3




    fInfo.baseName(); /// Tên file không có phần mở rộng - test
    QDateTime fileTime = fInfo.birthTime(); /// Thời gian  tạo file
  }







Lớp QDir

Lớp QDir cho phép bạn lấy thông tin và thư mục (thư mục), điều hướng qua hệ thống tệp và nhận danh sách các tệp trong một thư mục.


Bạn có thể lập trình truy cập thư mục bằng cách chỉ định đường dẫn đến thư mục một cách rõ ràng, từ lớp QFileInfo:








 QDir dir("C:\\TEMP");


  /// hoặc




  QFileInfo fInfo("C:/TEMP/result.txt");




  QDir tempDir = fInfo.dir();






 








Bạn có thể tìm ra đường dẫn đến một thư mục bằng cách sử dụng phương thức đường dẫn hoặc đường dẫn tuyệt đối. Bạn có thể lấy thư mục hiện tại của một chương trình đang thực thi bằng phương thức current (). Ví dụ sau tạo một biến lớp QDir, đặt nó vào thư mục hiện tại và hiển thị đường dẫn tuyệt đối cho thư mục đó trong một thông báo thông tin.








    QDir dir = QDir::current();
